View Auxiliary Port Configuration

Command: QA

Function: Displays the details of the Auxiliary port parameters.

Authorization: Not required.

Inputs: None

Outputs:
- IP Address
- Subnet Mask
- Default Gateway
- MAC address
- Ethernet speed settings

Errors: None
