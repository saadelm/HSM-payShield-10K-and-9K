Trace TCP/IP Route

Command: TRACERT

Function: Displays the route from the HSM to the specified address.

Authorization: Not required.

Inputs syntax: `tracert [options] host [packet size]`
Options include:
- `-t` (tos)
- `-v`
- `-w` (wait time)
- `-x`
