Test TCP/IP Network

Command: PING

Function: Tests the specified network route and the path to it.

Authorization: Not required.

Inputs: Standard ping syntax and parameters including:
- `-a`
- `-c`
- `-D`
- `-E`
- `-g`
- `-h` (host)
- `-I` (interface)
- `-i` (interval)
- `-l` (preload)
- `-L`
- `-n`
- `-o`
- `-P`
- `-p` (pattern)
- `-Q`
- `-q`
- `-R`
- `-r`
- `-s` (packet size)
- `-T` (ttl)
- `-t` (tos)
- `-v`
- `-w` (max wait)

Outputs: Results of the ping command.
